# JavaScript Objects, Scopes and Closures

JavaScript OOP tasks covering classes, inheritance, scopes, and closures using Node.js 14.x.

## Requirements
- Node.js 14.x
- semistandard style (Standard + semicolons + AirBnB)
- No `var` allowed

## Files

| File | Description |
|------|-------------|
| 0-rectangle.js | Empty Rectangle class |
| 1-rectangle.js | Rectangle with width & height |
| 2-rectangle.js | Rectangle with validation |
| 3-rectangle.js | Rectangle with print() |
| 4-rectangle.js | Rectangle with print(), rotate(), double() |
| 5-square.js | Square extending Rectangle |
| 6-square.js | Square with charPrint(c) |
| 7-occurrences.js | Count occurrences in a list |
| 8-esrever.js | Reverse a list without .reverse() |
| 9-logme.js | Log with call count using closure |
| 10-converter.js | Base converter using closure |
